# 🏦 ATM Banking System (Python)

## 📌 Overview
This is a Python-based ATM simulation program that allows users to perform basic banking operations like checking balance, depositing money, and withdrawing money.

The project demonstrates the use of loops and conditional statements to create a continuous and interactive menu-driven system.

---

## 🚀 Features
- Check account balance  
- Deposit money  
- Withdraw money  
- Input validation for transactions  
- Menu-driven system using loops  
- Exit option to stop the program  

---

## 🛠️ Technologies Used
- Python  
- Loops (while loop)  
- Conditional Statements (if-elif-else)  

---

## ▶️ How to Run

1. Install Python  
2. Download or clone this repository  
3. Run the program:

python main.py
